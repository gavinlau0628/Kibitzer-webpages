Kibitzer

ABOUT KIBITZER

Kibitzer is a search results aggregator for Craigslist. Based on "Search Profiles" created by users, it searches Craigslist and emails a report spreadsheet of what it found on a daily, weekly, or monthly basis. Kibitzerâ€™s Search Profile is a user generated set of parameters, containing information such as whitelisted words, blacklisted words, and the desired section of Craigslist. Once Kibitzer finds a search result that matches the Search Profile specs, it will add it to its database to later be sent to users that are subscribed to that Search Profile in the intervals described above.

GETTING STARTED

Signing Up

Signing up is easy, all you need to do is click on the Login button, and sign in with your Google Account.

Creating a Search Profile

Click on "Profile" on the navbar and it should lead you to a page that has "Create a Search Profile" at the top. If you need to, you can reset all your inputs by clicking the Reset button, otherwise once you are satisfied with the Search Profiles' specs, then you click the Submit button. You should now see your Search Profile in the tab titled "Your Search Profiles" below.

Subscribing to a public Search Profile

You can also browse for other users' Public Search Profiles that match what you want Kibitzer to search for, rather than creating a Search Profile yourself. You can start searching by clicking on "Browse" in the navigation bar. In the following page, type in whitelisted words that you want Kibitzer to search for and then click the Submit button. The application will then create a list of Search Profiles below. You can click the Subscribe button on those Search Profiles to receive notifications from them.

Unsubscribing from a Search Profile

If you don't want to get notifications about search Results from a Search Profile, you can unsubscribe by going to the "Profile" page and clicking unsubscribe on the Search Profile you don't want Search Results for. If you accidentally unsubscribe from a Search Profile, it will remain there and you can click "Subscribe" to subscribe to the Search Profile again.
